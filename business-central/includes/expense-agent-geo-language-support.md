Expense Agent is currently available in the US. It is validated and supported in English (EN-US) only. While it can be used in other languages, it might not function as intended. Language quality might vary based on the user's interaction or system settings, which might impact accuracy and the user experience. Learn more in [Copilot country/region availability and supported languages](../copilot-agents-region-language-availability.md).
