# Production Order - List (report)

The **Production Order - List** report provides an Excel-based overview of production orders, designed to support operational follow-up and analytical review. The report surfaces key master and scheduling information for production orders and allows users to filter the dataset to focus on specific time periods, locations, or order statuses.

The report is generated directly to Excel and includes two worksheets, each supporting a different analysis perspective.

Across the worksheets, the report displays information such as:
* Production order status (for example, Planned, Firm Planned, Released, Finished)
* Production order number
* Source type and source number
* Item number and description
* Routing number
* Location code
* Starting date
* Ending date
* Due date
* Quantity

This information enables you to review production orders across their lifecycle and correlate scheduling, sourcing, and item data in a single view.

## Worksheet: Prod. Order - List
This worksheet provides a consolidated list of production orders matching the selected filters. It is intended for operational overview and validation, such as:

* Reviewing which production orders exist in a given period
* Verifying order status and key dates
* Checking item, source, and location assignments
* Identifying upcoming or overdue production orders based on due dates

The layout is optimized for sorting, filtering, and ad-hoc analysis directly in Excel.

## Worksheet: Prod. Order Line 
This worksheet presents the same production orders with a focus on line-level and analytical breakdown, making it suitable for deeper inspection and comparison across orders.

Typical use cases include:
* Comparing production orders by item, location, or routing
* Analyzing production timelines across multiple orders
* Supporting planning and follow-up discussions with production or supply chain teams
