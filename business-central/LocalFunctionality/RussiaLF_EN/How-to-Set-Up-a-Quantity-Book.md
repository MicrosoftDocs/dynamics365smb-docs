# How to: Set Up a Quantity Book

Create a depreciation book to keep record of the fixed assets quantities that will be repaid at the time of release for the amount at which they were acquired.

 

## To set up a quantity book 

1. Choose the ![Search for Page or Report](https://github.com/DianaMalina/dynamics365smb-docs/blob/Pre-RussiaLF_EN/business-central/LocalFunctionality/RussiaLF_EN/search-icon.png) icon, enter **Depreciation Books**, and then choose the related link.
2. In the **Posting Book Type** field, select **Analytical**.
3. On the **Integration** FastTab, make sure that no options are selected.
4. Choose the ![Search for Page or Report](https://github.com/DianaMalina/dynamics365smb-docs/blob/Pre-RussiaLF_EN/business-central/LocalFunctionality/RussiaLF_EN/search-icon.png) icon, enter **FA Setup**, and then choose the related link.
5. In the **Quantitative Depr. Book** field, enter the code of the depreciation book that you created.
6. Choose the **OK** button.
